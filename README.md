# Quadratic equations solver

This module calculates the quadratic equation.

# How to use

Example of script launch on Linux, Python 3.5:

```bash
get_roots(a, b, c)

Example:
>>> get_roots(1, -2, 1)
(1.0, None)

```

# How to Unit test

```bash
$ python3 tests.py
```
